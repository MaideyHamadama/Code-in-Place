# Console Art Studio

A Python console application that lets you create small pieces of ASCII art interactively.

## Features

- Draw a graceful ASCII tree
- Render a spiral pattern
- Paint a rolling wave
- Build a word mosaic
- Show an inspirational quote
- Save the last generated art to a text file

## Run the app

```bash
python main.py
```

## Project structure

- `main.py` – entry point for the application
- `art_studio/app.py` – console menu and interaction logic
- `art_studio/generators.py` – art generation helpers

## Requirements

This project uses only the Python standard library.
